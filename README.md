# Tip Calculator App (Jetpack Compose)

## Overview
The **Tip Calculator App** is a simple and intuitive Android application built using **Jetpack Compose**. It helps users quickly calculate the tip amount based on a total bill and split the final cost among multiple people.

The app focuses on a clean UI, smooth interactions, and real-time calculations to enhance usability.

---

## Features

**Bill Input**  
  Enter the total bill amount easily.

- **Tip Percentage Slider**  
  Adjust the tip percentage dynamically using a slider.

- **Split Between Multiple People**  
  Increment or decrement the number of payers using **+ / - buttons**.

- **Real-Time Calculation**  
  Instantly calculates:
  - Tip amount  
  - Total amount per person  

- **Modern UI**  
  Built entirely with **Jetpack Compose** for a responsive and clean design.

---

## How It Works

1. Enter the **total bill amount**.
2. Use the **slider** to select the desired tip percentage.
3. Adjust the **number of people** sharing the bill.
4. The app automatically calculates:
   - **Tip Amount**
   - **Amount per Person**
